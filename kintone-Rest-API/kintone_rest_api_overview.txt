*. kintone auth:
X-Cybozu-Authorization: ('username:password') in base64

*. kintone app auth:

-. single app
X-Cybozu-API-Token: your token that generate from app

-. multiple app
X-Cybozu-API-Token: your token that generate from app1, your token that generate from app2
